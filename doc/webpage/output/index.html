4 August 2008

Release pyhdf-0.8-1.

Welcome to PyHDF, a Python interface to the NCSA HDF4 library. The following subset of HDF4 components are supported:

  • SD (Scientific Dataset)
  • VS (Vdata)
  • V (Vgroup)
  • HDF (common declarations)

HDF5 files can be read using PyTables or h5py.
