pyhdf  - a python interface to the NCSA HDF4 library

pyhdf is a python interface to the NCSA HDF4 library. Among the numerous components offered by HDF4, the following are currently supported by pyhdf :

  • SD (Scientific Dataset)
  • VS (Vdata)
  • V (Vgroup)
  • HDF (common declarations)

Current version :

  • pyhdf-0.7-3,  July 12 2005

Requirements :

  • pyhdf has been successfully tested with HDF versions up to 4.2r1. Note that there exists a version 5 of HDF, which is totally incompatible with version 4. pyhdf does not work with HDF5.
  • Numerical Python version 22 or above is required.
  • Python v2.2 and above is required.
  • pyhdf should work on most Unix/Linux systems where the above software can be installed. It has been installed (among others) on the following environments :Linux (kernel 2.4.19 and above), Tru64 4.0.f and above, Solaris 8, AIX 4.
