pycdf - A python interface to Unidata netCDF library

pycdf is a python interface to the Unidata netCDF library.  It provides an almost complete coverage of the netCDF C API, wrapping it  inside easy to use python classes. 

netCDF arrays are handled using array objects provided either by the python Numeric or numarray packages. The Numeric and numarray packages are closely related, and are hosted on the same SourceForge project, from which the source code can be downloaded. Ready-made binary distributions are available for most Unix/Linux environments. 

Current pycdf version :

  • pycdf-0.6-0,  January 02 2006    NEW!

New with the 0.6-0 release:
  • numarray support
  • netCDF large file support (CDF2 64-bit format), which appeared in netCDF 3.6
  • ellipsis notation in slices
  • validity checks in array assignments, scalar broadcasting
  • improved documentation
  • many bug fixes

Requirements :

  • Python v2.2 and above.
  • netCDF library versions up to 3.6.1.
  • Numeric version 22 or above. 
  • numarray can also be used. The only numarray version tested so far is 1.3.3.
  • pycdf should work on most Unix/Linux systems where the above software can be installed.It has been installed (among others) on the following environments : Linux (kernel 2.4.19 and above), Tru64 4.0.f and above, Solaris 8, AIX 4.
