pycdf - A python interface to Unidata netCDF library

pycdf is a python interface to the Unidata netCDF library.  It provides an almost complete coverage of the netCDF C API, wrapping it  inside easy to use python classes. netCDF arrays are handled using Numerical Python array objects.

Current version :

  • pycdf-0.5-4,  August 15 2005    NEW!

Requirements :

  • pycdf has been successfully tested with netCDF versions up to 3.5.0. pycdf has not been tested yet with release 3.6.
  • Numerical Python version 22 or above is required.
  • Python v2.2 and above is required.
  • pycdf should work on most Unix/Linux systems where the above software can be installed.It has been installed (among others) on the following environments : Linux (kernel 2.4.19 and above), Tru64 4.0.f and above, Solaris 8, AIX 4.
